Capstone Project

Home Design App

The Home Design App is an Android application that allows users to explore, purchase, and visualize furniture in their homes using Augmented Reality (AR) technology. The app provides users with a rich experience by integrating Firebase for authentication, Firestore for real-time data management, and AR for immersive furniture placement.

Objective: To create a comprehensive home design solution where users can explore and visualize furniture in their own spaces using AR.
Skills Demonstrated: Android development (Kotlin), Firebase integration, Firestore real-time database, Augmented Reality (AR), and UI/UX design.
Challenges: Managing real-time data updates for multiple users and implementing smooth AR functionality were significant challenges, solved using Firebase's real-time capabilities and ARCore integration.
Significance: This project demonstrates my expertise in building a feature-rich Android application with real-time data and augmented reality, culminating my learning in the Mobile Application Development (MAPD) program.
